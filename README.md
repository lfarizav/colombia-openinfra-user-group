# Colombia OpenInfra User Group — Website

Public website for the Colombia OpenInfra User Group community, deployed on **GitHub Pages** at:
**https://lfarizav.github.io/colombia-openinfra-user-group/**

## Architecture

Mirrors the [beanters.com](https://beanters.com) no-build React pattern:

| File | Purpose |
|---|---|
| `index.html` | Entry point: SEO meta, JSON-LD, inline CSS, React mount |
| `data.js` | Site content (`window.SITE`): events, speakers, socials — **auto-regenerated by Heritage** |
| `js/app.js` | All React components (no JSX, no build step) |
| `react*.production.min.js` | Local React 18 UMD bundles |
| `assets/` | SVG logos and cover graphics |
| `presentations/` | Speaker slide PDFs |
| `tests/` | Playwright smoke tests |

## Content management

Content is managed from the **Heritage community control panel** (`http://localhost:5000/community`).
Clicking **Publish** regenerates `data.js` in this repo. Then commit and push:

```bash
git add data.js
git commit -m "chore: update site content"
git push
```

## Local preview

```bash
python3 -m http.server 8088
# open http://localhost:8088/
```

## QA

```bash
npm install
npx playwright test          # smoke: nav, social links, sections
```

## Enabling GitHub Pages

In the repository settings → Pages → Source: **Deploy from branch** → `main` / root (`/`).
